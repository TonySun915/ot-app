📱 一、iPhone 安裝方式（Safari 加入主畫面）
安裝步驟:
  - 使用 Safari 開啟系統網址（⚠️ 請勿使用 LINE、Chrome 或其他瀏覽器）
  - 點擊 Safari 底部的 「分享」 按鈕（⬆️）
  - 選擇 「加入主畫面」
  - 確認名稱後，點擊 「加入」

🤖 二、Android 安裝方式（Chrome 加入主畫面）
安裝步驟:
  - 使用 Chrome 瀏覽器 開啟系統網址
  - 畫面上方或選單中會出現 「加入主畫面」 提示
  - 若未自動顯示，請點擊右上角「⋮」
  - 選擇 「加入主畫面」
  - 確認後加入

🔔使用說明
- 桌面會出現「加班費試算」圖示
- 點擊後將以 全螢幕 App 模式 開啟
- 第二次之後可在 離線狀態 使用
